Projects

Supreme Court Case Semantic Search and Chat Bot

June - July 2023

I made a chatbot knowledgeable in constitutional law. This was done by creating vector embeddings of the contents of all supreme court rulings and storing them in a Chroma vector database. This allowed a simple chatbot to answer questions with context given by the cases.

MNIST Classification Neural Network Without ML Libraries

December 2022

I developed a multilayer perceptron neural network to classify handwritten digits without using premade machine learning libraries. Instead the I used used the Numpy library for linear algebra. This model had an accuracy of 0.96. The model was trained on the MNIST dataset.

Genre Classification

November - December 2022

Using a random forest model developed in R, I classified songs into 6 genres using features provided by the Spotify API. The data provided by the Spotify API details subjective musical attributes of the songs with catagories like danceability.

Anonymize

October 2022

I implemented a real-time face blurring software with Gaussian blur, pixelation and blackout methods of obscuration. I used OpenCV for facial detection and masking effects. This project was done as some practice with OpenCV.

Violet Light

October 2022

Sus Foods

November 2021

The Buzz

September - December 2021

I created a full stack social media app including web and mobile front-ends, server backend, and admin cli. The backend was made in Java using the Spark framework and used a PostgreSQL database. Web and mobile front-ends were written in Typescript using React and React-Native respectively with Material UI. This was developed in a 5 person team using Scrum methodology.
